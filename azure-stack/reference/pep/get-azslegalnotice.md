# Get-AzSLegalNotice

## Synopsis
Gets the legal notice caption and text.

## Syntax

```
Get-AzSLegalNotice
```

## Description
Gets the legal notice caption and text, if it was previously set by the [Set-AzSLegalNotice](set-azslegalnotice.md) cmdlet.

## Examples

### Example 1
```
Get-AzSLegalNotice 
```
